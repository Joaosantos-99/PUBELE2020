    Introdução

    O intuito deste trabalho é o desenvolvimento de um ficheiro HTML com base no conteúdo de um ficheiro XML. Neste ficheiro XML está presente o conteúdo de três relatórios, com estruturas semelhantes, tendo sido desenvolvidos processos/funções que permitissem a realização do trabalho.

    Desenvolvimento

    1. Criou-se um ficheiro xml com todo o corpo dos relatórios pretendidos;
    2. Realizou-se na linha de comandos a operação xmllint para verificar se o ficheiro XML era válido;
    3. Criou-se um ficheiro dtd de maneira a validar as tags do xml com a seguinte linha de código ' xmllint -dtdvalid report.dtd xml1.xml';
    4. Iniciou-se código python capaz de passar todo o conteudo do ficheiro XML para HTML;
    5. Geraram-se inicialmente duas funções com a estrutura pretendida para os relatórios e para a criação da página de índice;
    6. Implementaram-se funções capazes de extrair a informação do XML;
    7. Na função main aplicam-se as funções criadas anteriormente, por forma a preencher o template;
    8. Após a conclusão de todas as funções correu-se o programa na linha de comandos através da operação 'python3 html2.py';
    9. Observou-se os resultados pretendidos, e apenas se alterou o design do ficheiro HTML;
    10. De maneira a conseguir correr os ficheiros o que aconselhamos é que faça o download de todos os ficheiros em formato 'png', 'xml' e 'py'. Em seguida deverá abrir na linha de comandos a pasta onde guardou estes ficheiros e depois realizar a operação 'python3 html2.py'. Deste modo, os ficheiros html serão gerados na pasta onde tem os restantes ficheiros que já tinha transferido;
    11. Para abrir apenas terá de clicar nos mesmos e assim irá abrir um separador no browser com o mesmo. Deve abrir o ficheiro com nome 'index.html' uma vez que esse redireciona para os restanteste ficheiros;
